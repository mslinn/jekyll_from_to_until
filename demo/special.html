I had a problem with a filter that needed a special character. The page was {% href label='mslinn.com/jekyll_plugins/jekyll_pre.html#comprehensive_example' url='https://mslinn.com/jekyll_plugins/jekyll_pre.html#comprehensive_example' %}.

The solution I chose was to allow special characters to be specified as HTML entities. For example, the closing squiggly bracket, }, can be encoded as &#x7d; or as &rcub;.

{% capture css %}{% flexible_include '$msp/_sass/jekyll_pre.scss' %}{% endcapture %}

Using an HTML Entity

{% pre dedent label="Jekyll Filter Invocation" %} {% raw %}{{ css | from: '.error' | to: '&rcub;' | strip }}{% endraw %} {% endpre %} {% pre dedent label="Resulting CSS Fragment" %} {{ css | from: '.error' | to: '}' | strip }} {% endpre %}

Using an HTML Number

{% pre dedent label="Jekyll Filter Invocation" %} {% raw %}{{ css | from: '.error' | to: '&#x7d;' | strip }}{% endraw %} {% endpre %} {% pre dedent label="Resulting CSS Fragment" %} {{ css | from: '.error' | to: '}' | strip }} {% endpre %}
